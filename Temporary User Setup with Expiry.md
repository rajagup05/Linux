
## Temporary User Setup with Expiry

`Task:` **Create a user named `ammar` on App Server 2. Set the expiry date to `2026-12-07`, ensuring the user is created in lowercase as per standard protocol.**

- used `ssh user2@server2` to connect/login into server2.
- used `sudo useradd ammar -e 2026-12-07` to set an expiry date while creating a user.
