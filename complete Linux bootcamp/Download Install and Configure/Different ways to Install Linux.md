
## Different ways to Install Linux

### 1. USB

This is the standard consumer method. You write a Linux image directly to a thumb drive to create a bootable storage medium.

### 2. ISO image

You can boot an ISO file directly from your local hard drive without burning it to an external medium.

### 3. Network Boot

Best for Multiple Computers in a enterprise environment where required files are installed from a shared server.

### 4. CD/DVD

The legacy approach for older hardware that cannot boot from USB.
