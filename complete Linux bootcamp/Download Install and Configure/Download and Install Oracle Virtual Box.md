
## Download and Install Oracle Virtual Box

Downloading and installing Oracle VM VirtualBox is a straightforward process:

- Go to the ⁠Official VirtualBox Downloads Page.
- Choose your operating system's installation package (e.g., Windows hosts).
- Run the installer and proceed through the setup using the default settings.

### step by step installation

**1. Download the installer**

- Open your web browser and navigate to the VirtualBox Official Website.
- Click the large "Download VirtualBox" button on the homepage.
- Select the installer that matches your operating system (Windows, macOS, Linux, or Solaris). For example, if you are using a PC, select "Windows hosts" to download the .exe file.

**2. Run the setup wizard**

- Locate the downloaded file in your computer’s Downloads folder and open it.
- A User Account Control (UAC) prompt may appear asking for permission to make changes. Click Yes.
- The Setup Wizard will open; simply click Next to proceed.

**3. Configure setup options**

- **License Agreement**: Review the terms, select "I accept the terms in the License Agreement," and click Next.
- **Custom Setup**: You can safely leave all custom setup options (such as locations, shortcuts, and USB support) at their default settings and click Next.
- **Network Interfaces Warning**: You may see a prompt warning you about a temporary network disconnection. This is normal and occurs when VirtualBox sets up its host network adapters. Click Yes to continue.
- **Missing Dependencies**: Depending on your system (such as missing Python core components on Windows), the installer may ask to install additional dependencies. Click Yes to allow them.

4. Complete the installation

- Click Install to begin the actual installation process.
- Once the installation completes successfully, leave the "Start Oracle VM VirtualBox" checkbox checked.
- Click Finish to launch the Oracle VM VirtualBox Manager immediately.
