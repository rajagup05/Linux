
## Take a snapshot of a VM running in Oracle Virtual Box

To capture the current state of a running VM (so you can easily revert if something goes wrong), click Machine in the top menu bar of your Virtual Machine window and select Take Snapshot. Give your snapshot a recognizable name and description, then click OK.



For a full snapshot of the VM's exact data, memory, and settings at this very moment:

- While your VM is running, click Machine > Take Snapshot at the top left of the VM's display window.
- Alternatively, use the shortcut: Press the Host Key + T (The Host Key is typically the Right Ctrl key for Windows/Linux or Command key for Mac).
- Type a Name and Description (e.g., "Before Software Update").
- Click OK
