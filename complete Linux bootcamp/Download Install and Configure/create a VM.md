
## create a VM in Oracle Virtual Box

To create a new virtual machine (VM) in ⁠Oracle VirtualBox, open the VirtualBox Manager and click the New button. This launches a guided wizard that walks you through configuring your new virtualized environment.

### step-by-step process

**1. Name and Operating System:**

- **Open Wizard**: Click Machine > New or press Ctrl + N.
- **Name**: Type a descriptive name for your VM.Folder: Keep the default path or choose where to store the VM files.
- **ISO Image**: Select the operating system ISO file you downloaded (e.g., Windows or Ubuntu).

**2. Allocate Hardware resources:**

- **Base Memory (RAM):** Choose how much RAM to allocate using the slider. Stay within the green zone to prevent slowing down your host computer.
- **Processors (CPU):** Assign at least 2 CPU cores for smooth performance if your host hardware allows.

**3. Create a Virtual Hard Disk:**

- **Disk Setup:** Choose Create a Virtual Hard Disk Now.
- **Disk Size:** Allocate enough space based on the guest operating system's requirements (e.g., 30–50 GB or more).
- **Allocation Type:** By default, VirtualBox uses Dynamically allocated disks, which only consume real hard drive space on your host machine as the VM fills up.

**4. Finalize the VM:**

- **Finish**: Review the configuration matrix on the summary screen and click Finish.
- **Launch**: Select your new VM from the left-side panel and click the green Start arrow on the top toolbar.
- **Install OS:** The VM will boot from your selected ISO image. Follow the on-screen prompts of the target operating system to finish its installation.
