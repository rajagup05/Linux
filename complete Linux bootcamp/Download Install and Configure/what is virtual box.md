
## what is virtual box

VirtualBox is free, open-source virtualization software that lets you run multiple operating systems (like Windows, Linux, or macOS) simultaneously on a single computer. It acts as a "computer within a computer," creating an isolated virtual environment so you can test software or run older programs.

### How it works ?

How it WorksVirtualBox uses the hardware resources of your actual computer (your Host OS) to create a "Virtual Machine" (your Guest OS). You simply assign a portion of your processor (CPU), memory (RAM), and hard drive space to the virtual machine.

Once set up, you can start the virtual machine like a normal app, and it will boot up and display its own desktop. You can use it to browse the web, install programs, and run files exactly as you would on a physical machine, without affecting your main operating system.

### Why people use it ?

- **Run Incompatible Software**: Use applications or older games designed for a different operating system without needing a separate physical device.
- **Safe Testing**: Experiment with new or unknown operating systems (e.g., trying a new Linux distribution) or test risky software safely. If something goes wrong, you can simply delete and recreate the virtual machine without harming your primary computer.
- **Development**: Software developers use it to easily build, compile, and test code across multiple platforms.

### Common use cases

- **Running Linux on Windows**: A developer who primarily uses Windows but needs a Linux environment for programming.
- **Legacy Support**: Running an older operating system, like Windows 7, to run old business programs or legacy PC games that aren't supported on modern systems.
- **Safe Web Browsing**: Opening questionable email attachments or visiting untrusted websites inside the isolated virtual environment to prevent malware from infecting your main computer.

### Supported OS


- **Host Operating Systems (Where you install VirtualBox):** `Windows`, `macOS`, `Linux`, and `Solaris`.
- **Guest Operating Systems (What you can run inside):** Virtually any modern OS, including `Windows (11, 10, 8, 7)`, hundreds of Linux distributions (`Ubuntu`, `Fedora`, `Debian`), `macOS`, and more.
