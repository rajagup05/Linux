
## history of linux

Linux began in 1991 as a personal project by Finnish student Linus Torvalds. Frustrated by the high cost of Unix and the limitations of Minix, he set out to build a free, open-source operating system kernel for his personal computer.

### The GNU connection and Open source

- **The GNU Project**: Richard Stallman's GNU project had created many essential system tools but lacked a functional kernel. By combining the Linux kernel with GNU utilities, developers created a complete, fully functional operating system.
- **Going Open Source**: Initially, Linux code could not be used commercially. Following a 1992 transition, Linus relicensed the kernel under the GNU General Public License (GPL), guaranteeing the software would remain free and open to modification.
