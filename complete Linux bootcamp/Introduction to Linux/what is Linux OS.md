
## what is Linux OS 

Linux is a free, open-source operating system that manages computer hardware and software. Created by Linus Torvalds in 1991, it powers everything from personal laptops and cloud servers to smartphones (via Android) and supercomputers.

### How linux works

- **The Kernel**: The core of Linux that manages your CPU, memory, and storage. It is the bridge between the physical hardware and the programs running on it.
- **Distributions (Distros):** Because Linux is open-source, developers bundle the kernel with different applications, desktop interfaces, and tools to create ready-to-use versions called "distributions". Popular examples include Ubuntu, Linux Mint, and Fedora.

### why people use it

- **Free and Open Source:** You can legally download, view, and modify the source code to fit your exact needs.
- **Highly Secure:** Due to its permission models and massive community constantly patching bugs, it is highly secure and rarely requires third-party antivirus software.
- **Customizable**: You aren't locked into a single layout; you can completely change how the graphical interface and system menus look.
- **Performance**: Linux runs incredibly well on older or less powerful hardware, which is why it dominates cloud servers and IoT devices.
