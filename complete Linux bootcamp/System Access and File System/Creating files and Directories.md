
## Creating files and Directories

In Linux, you can create files and directories using standard terminal commands like touch for files and mkdir for directories.

### Creating Directories (Folder)

The mkdir (make directory) command is used to create one or multiple folders.

- single directory: `mkdir my_folder`
- multiple directories at once: `mkdir folder1 folder2 folder3`

### Creating files

Linux offers several ways to create files, depending on whether you want them empty or with text content.

- creating empty file: `touch document.txt`
- multiple files at once: `touch index.html style.css script.js`
- using redirection: `echo "Hello World" > hello.txt`
- using `cat` command: `cat > note.txt`
- using editors like `nano`: `nano report.md`
