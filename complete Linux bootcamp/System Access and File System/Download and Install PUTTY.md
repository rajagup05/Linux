
## Download and Install PUTTY

### Download the Installer:

- Navigate to the official PuTTY Landing Page and click the primary download link to reach the latest release section.
- Select the correct package under the Package files section. For most modern Windows 10 and Windows 11 computers, click on the 64-bit x86: putty-64bit-0.84-installer.msi link.
- Alternative: If you prefer an app deployment platform, you can also securely download and install the official package directly through the Microsoft Store PuTTY Page.


### Run the Installation Wizard

- Open your default Downloads directory and double-click the newly downloaded .msi installer file.
- Advance through the standard prompts by clicking Next on the initial Welcome setup screen.
- Confirm the installation location path, which defaults safely to C:\Program Files\PuTTY\, and click Next.
- Enable any optional software features on the project customization window, such as selecting the toggle to add a PuTTY shortcut directly to your desktop.
- Finalize the setup by clicking Install, authorizing any administrative security prompts that appear, and clicking Finish

Once completed, you can quickly locate and launch the desktop emulator app through your Windows Start Menu.
