
## monitor users

To monitor users on a Linux system, the most effective dynamic command is w, which immediately shows who is currently logged in, their active terminal, idle time, and the exact process or command they are running.

### Active User Monitoring:

- `w`: Displays logged-in users, terminal types, login times, idle times, and their current command lines.
- `who`: Lists currently logged-in users along with their assigned terminal lines and connection times.
- `users`: Outputs a short, space-separated list of raw usernames active on the system.
- `last`: Parses system logs to display a comprehensive history of recent user logins, logouts, session durations, and system reboots.
