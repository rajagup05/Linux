
## system utility commands

System utility commands in Unix/Linux are built-in tools used to check system status, view dates, identify hardware, and perform quick calculations.

### date

Displays or sets the system date and time.

Example: `date "+%Y-%m-%d %H:%M:%S"`

### uptime

Shows how long the system has been running.

Example: `hostname -i` (Displays the network IP address)

### uname 

Prints detailed system and kernel information.

Common Flag: `uname -a` (Prints all available system details)
