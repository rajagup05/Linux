
## top command

The top command in Linux is a native, real-time system monitoring utility that displays a dynamic view of running processes and resource utilization. It serves as a command-line task manager, prioritizing and sorting processes by resource consumption so administrators can instantly identify bottlenecks.

### few commands:

- `top -u [username]`: Restricts the display to processes owned by a specific user.
- `top -p [PID]`: Monitors only a designated Process ID.

### key shortcuts: 

- `q`: Quits the utility and returns to the prompt.
- `h`: Opens the interactive help screen menu.
- `P` (`Shift + P`): Sorts the entire process table by CPU usage (default).
- `M` (`Shift + M`): Sorts the entire process table by Memory usage.
- `k`: Kills a process by prompting you for its PID and signal type.
